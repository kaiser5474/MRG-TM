recorrido

Up 
GET /NRecorrido_select
Select data from the 'N_Recorrido' table. Data are returned in JSON format.

Query parameters

fields
Query Parameter —
where
Query Parameter — An optional WHERE clause. E.g. city='boston'. Can also use arguments in where clause. e.g. city=:c_city
order
Query Parameter — An optional ORDER BY clause
arguments
Query Parameter — A comma delimited list of argumentname/value pairs. Syntax for argument name is dataType_name, e.g. T_DOB is an argument of time value. Example t_dob=1972/12/6,c_place=medford
pagesize
Query Parameter — Specifies number of records per 'page'. If blank, all records are returned.
pageNumber
Query Parameter — Specifies the page of data to return. Only meaningful if pagesize is not blank.

Responses

200

Successful request.

Curl Example

curl "{{host}}{{basePath}}/NRecorrido_select?fields={fields}&where={where}&order={order}&arguments={arguments}&pagesize={pagesize}&pageNumber={pageNumber}"
